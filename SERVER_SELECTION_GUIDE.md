## 安全配置

### 购买后必做

#### 1. 修改SSH端口（防止暴力破解）

```bash
# 登录服务器
ssh root@your-server-ip

# 备份配置
cp /etc/ssh/sshd_config /etc/ssh/sshd_config.bak

# 修改端口
vi /etc/ssh/sshd_config
# 找到 #Port 22，改为 Port 2222

# 重启SSH服务
systemctl restart sshd
```

#### 2. 配置防火墙

```bash
# Ubuntu 20.04
ufw allow 2222/tcp    # SSH端口
ufw allow 80/tcp      # HTTP
ufw allow 443/tcp     # HTTPS
ufw allow 3000/tcp    # 后端端口
ufw enable
```

#### 3. 禁用root远程登录

```bash
# 创建普通用户
adduser admin
usermod -aG sudo admin

# 修改SSH配置
vi /etc/ssh/sshd_config
# 添加：PermitRootLogin no

# 重启SSH
systemctl restart sshd
```

#### 4. 安装fail2ban（防暴力破解）

```bash
apt install fail2ban
systemctl enable fail2ban
systemctl start fail2ban
```

---

### 腾讯云安全组配置

登录腾讯云控制台：
1. 进入「轻量应用服务器」→「防火墙」
2. 添加规则：

| 协议 | 端口 | 来源 | 说明 |
|-----|------|------|------|
| TCP | 22（或修改后的端口） | 0.0.0.0/0 | SSH |
| TCP | 80 | 0.0.0.0/0 | HTTP |
| TCP | 443 | 0.0.0.0/0 | HTTPS |
| TCP | 3000 | 127.0.0.1 | 后端（仅本地） |

**注意**：不要开放 3000 到公网，通过 Nginx 代理

## 性能优化建议

### 1. 开启CDN加速

推荐使用：
- 腾讯云 CDN
- 阿里云 CDN

**适用资源**：
- 静态文件（图片、CSS、JS）
- 床位图片

**节省带宽**：可节省 60-80% 带宽费用

---

### 2. 数据库优化

```sql
-- 连接数
max_connections = 100

-- 缓冲池（根据内存调整）
innodb_buffer_pool_size = 512M  # 2核4GB推荐
```

---

### 3. 启用Gzip压缩

Nginx 配置：

```nginx
gzip on;
gzip_types text/plain text/css application/json application/javascript;
gzip_min_length 1000;
```

## 常见问题

### Q1: 首次选择哪个配置？

**A**: 推荐**腾讯云轻量应用服务器 2核4GB配置**，首年约¥100，完全够用。

### Q2: 域名必须购买吗？

**A**: 可以先使用 IP 地址访问，但小程序必须使用域名。建议购买一个 .com 域名，一年约¥60。

---

### Q3: 可以使用学生优惠吗？

**A**: 可以！腾讯云和阿里云都有学生优惠，认证后可以享受 ¥10-30/月的超低价。

---

### Q4: 服务器部署后如何访问？

**A**:
1. 通过 SSH 登录服务器：`ssh root@your-ip`
2. 安装 Node.js、MySQL
3. 上传代码并启动服务
4. 配置 Nginx
5. 通过浏览器访问域名

---

### Q5: 需要备案吗？

**A**: 如果域名指向国内服务器，需要ICP备案。备案流程约7-20天。如果不想备案，可以使用香港或国外服务器。

---

### Q6: 可以升级配置吗？

**A**: 可以随时升级CPU、内存、带宽，按天计费。

---

### Q7: 如何节省成本？

**A**:
1. 一次购买3年
2. 新用户充分利用首年优惠
3. 使用学生优惠
4. 开启CDN节省带宽
5. 定期清理无用数据
